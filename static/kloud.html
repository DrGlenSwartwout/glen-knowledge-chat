Simple to use, without Bluetooth

  1. Switch the unit on with the power button. The blue power light comes on.
  2. Once it warms up, the green check light comes on.
  3. Select a program (1 to 5) with the right and left arrow button.
  4. Press start. The selected program light flashes while the program runs.
  5. The unit beeps at the end of the 15 minute session. Turn the toggle off to save the battery.

We recommend using intensity level 1 or 2 for most sessions. Bluetooth is not a therapeutic signal and can stay off; it is only used to load software updates from the company.

Controls at a glance

  • On / Off toggle. Circle is off, line is on. On power up the unit beeps, the power light turns blue, and the check light turns green.
  • Start / Stop (triangle and square). Press to start or stop a program. It stops automatically after 15 minutes.
  • Set intensity. While a program is running, press and hold Start / Stop for about 2 seconds, then use the right and left button to cycle intensity 1 to 5. Intensity is set separately for each program.
  • Select program (two arrows). When a program light is steady, press the right and left button to change the selection.
  • Check battery. Press and hold the right and left button for about 3 seconds. P1 to 5 lit is 100 percent, down to P1 only at 10 percent. A full charge takes about 4 hours.
  • Charging. Turn the toggle off and plug in the cable. The charge light is blue while charging and the cable light turns green when full.
