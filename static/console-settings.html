Active Write Mac Live

Designates which Mac is allowed to run scripts that mutate shared state (SQLite DBs, synced JSON, ingestion writers). Scripts on the other two Macs will fail fast with a clear message.

Why this exists: the AI-Training vault syncs across 3 Macs via Obsidian Sync, but the local SQLite DBs (chat_log.db, e4l.db, ingredients.db) are excluded from sync. If two Macs ran the same writer simultaneously, the local DBs would diverge silently.
